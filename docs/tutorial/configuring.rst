=============================================
writing scripts and YAML config files for HVC
=============================================

To encourage reproducibility, `hybrid-vocal-classifier` is configured either with
scripts or YAML configuration files. This way you create a concrete record
of the approach you have used, so that you have it for later and so that you can
share it with others.
